# Flighthour Landing Pages

Marketing-Landing-Pages für [Flighthour.de](https://flighthour.de) - Flugsimulator-Erlebnisse als unvergessliche Geschenke.

## 🎯 Über das Projekt

Diese Repository enthält spezialisierte Landing Pages für verschiedene Marketing-Kampagnen von Flighthour. Die erste Landing Page fokussiert sich auf Flighthour-Erlebnisse als Geburtstagsgeschenke für Jung und Alt.

## 🚀 Technologie-Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animationen**: Framer Motion
- **TypeScript**: Für Type-Safety
- **Deployment**: CapRover

## 📋 Entwicklung

```bash
# Installation
npm install

# Entwicklungsserver starten
npm run dev

# Production Build erstellen
npm run build

# Production Server starten
npm start
```

## 🎨 Design-System

Das Design basiert auf der Corporate Identity von Flighthour.de:
- **Primärfarbe**: Gelb (#FBB928)
- **Sekundärfarbe**: Schwarz (#000001)
- **Schriften**: ITC Avant Garde Gothic & Poppins

## 📱 Landing Pages

### 1. Geburtstagsgeschenk-Idee
- **URL**: `/` (Homepage)
- **Zielgruppe**: Menschen, die ein besonderes Geburtstagsgeschenk suchen
- **Features**: 
  - Emotionale Hero-Section
  - Zielgruppen-spezifische Inhalte (Jung & Alt)
  - Paket-Übersicht mit CTAs
  - Testimonials
  - FAQ-Bereich
